# Mini Scientific Calculator

A simple command-line calculator built using Python. This project performs basic mathematical operations and is beginner-friendly.

## ✨ Features
- ➕ Addition
- ➖ Subtraction 
- ✖️ Multiplication
- ➗ Division
- Handles invalid inputs with error messages
- Easy to run from terminal

## 🚀 How to Run
1. Make sure Python 3.x is installed
2. Clone this repository or download calculator.py
3. Open terminal and run:
```bash
python calculator.py
